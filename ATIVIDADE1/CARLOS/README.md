# Carlos Aplewicz

## Objetivo da Atividade

O objetivo desta atividade é demonstrar a execução de um programa simples na linguagem Dart, verificando a instalação do ambiente de desenvolvimento e a capacidade de compilar e executar código básico no terminal.

## Instruções para Executar o Programa
Instale o Dart (caso ainda não tenha).

Crie um arquivo com o nome:

`hello.dart`

Dentro do arquivo, adicione o seguinte código:

`void main() {
  print('Hello, World!');
}`

Abra o terminal na pasta onde está o arquivo.

Execute o programa com o comando:

`dart run hello.dart`

Exemplo de Saída no Terminal

`Hello, World!`
