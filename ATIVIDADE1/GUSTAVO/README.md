## Objetivo da atividade
O objetivo desta atividade foi nada mais do que fazer um script simples em linguagem Dart em que será exibido no terminal a famosa mensagem "Hello World"

## Instruções para executar o script
1. Instale o Dart SDK na máquina
2. Crie um arquivo chamada `main.dart`
3. Copie e cole o seguinte código no arquivo:
```
void main() {
    print("Hello World");
}
```
4. Abra o terminal na pasta onde o arquivo foi salvo
5. Executa o comando abaixo:
```
dart run main.dart
```
